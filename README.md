# SavvyIndians - Creative AI Services

A modern, futuristic website for SavvyIndians, a creative AI startup that combines cutting-edge AI technology with creative expertise to deliver exceptional results.

## Features

- **Modern Design**: Futuristic UI with glass morphism effects and animated backgrounds
- **AI Services Showcase**: Four main services with detailed descriptions
  - AI Clone Personalized Avatar
  - Realistic AI Generation  
  - Custom Styling
  - Multiple Formats & Commercial License
- **Masterclass Signup**: Email capture form for AI creativity masterclass
- **Contact Integration**: Direct contact with phone and email
- **Responsive Design**: Fully responsive across all devices
- **Smooth Animations**: Custom Tailwind CSS animations and transitions

## Technologies Used

- **Next.js 15** - React framework with App Router
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first CSS framework
- **React Hooks** - Modern React state management

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) to view the website

## Project Structure
